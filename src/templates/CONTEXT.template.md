# CONTEXT.md — OKR Cycle Narrative & Memory

> The durable narrative companion to `STATE.md`. Where `STATE.md` holds the
> structured record (the *what*), `CONTEXT.md` holds the reasoning (the *why*):
> assumptions, research, decisions, and rationale. OKR Core reads this at session
> start to rehydrate context and appends to it as the cycle unfolds. This is the
> primary defense against context rot across sessions.
